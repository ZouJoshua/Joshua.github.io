关于作者

92年猴，15年毕业（统计学），些微完美主义，些微强迫症。

初入职场的小学僧一脚趟进数据科学圈，希望在这里找到属于自己的一亩三分地。

目前，从事数据挖掘分析，喜欢钻研一些技术问题，日常研究数据挖掘、机器学习等多种算法。

主要语言：Python、R

涉猎算法：传统统计算法、数据挖掘算法、机器学习算法

数据库：Oracle、MySQL、Mongodb、Redis

操作系统：Windows、Linux

关于本站

开通于2017年5月22日,有些功能不够完美,仍在不断完善中。

本站所有文章除非特别说明均属原创，转载需保留署名和原文链接。

本站使用基于 Node.js 的 Hexo 静态博客框架开发，部署于 Github 。

本站所有博客纯属个人笔记，错误、描述不到位、理解片面等等情况在所难免，如有不正确的地方欢迎评论中指正，我会及时更正，同时也欢迎大家相互交流。因为博客建立的太晚，很多以前写的笔记我需要重新整理分享在这里，所以你会看到很多很基础很入门的文章。

顺便吐槽一下自己，真的是太懒了，懒到自己都嫌弃自己的地步，一年多以前就有做一个个人网站的想法，一直拖到现在。
